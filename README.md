# Mobile Automation Framework

This project is a mobile automation framework built using **Java**, **Appium**, and **Cucumber**.

## 📁 Project Structure

* `src/main/java` → Utility classes and core logic
* `src/test/java` → Test classes and step definitions
* `src/test/resources` → Feature files and configuration
* `pom.xml` → Maven dependencies and build configuration

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/vsuyash/mobile-automation-feb-2025.git
   ```

2. Navigate to project:

   ```bash
   cd mobile-automation-feb-2025
   ```

3. Run tests:

   ```bash
   mvn clean test
   ```

## ⚙️ Requirements

* Java 8+
* Maven
* Appium Server
* Android Studio / Emulator or real device

## 🧪 Features

* Appium-based mobile automation
* Cucumber BDD framework
* Reusable utility classes

## 📌 Notes

* Update `capabilities.json` as per your device/emulator setup
* Ensure Appium server is running before executing tests
